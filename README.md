# Advanced EDA Project

An automated Exploratory Data Analysis (EDA) tool designed to streamline data visualization. This project generates multiple statistical plots from input datasets to help identify patterns, trends, and correlations instantly.

## 🚀 Features

The tool automatically generates the following advanced statistical visualizations:

* **Pairplot:** Visualizes pairwise relationships across the entire dataset.
* **Heatmap:** Displays correlation matrices to find linear relationships.
* **Boxplot:** Highlights data distribution, medians, and outliers.
* **Violin Plot:** Combines box plots with density traces to show data structure.
* **Regression Plot:** Fits and visualizes linear trends between variables.
* **KDE Plot:** Smooths out continuous probability density distributions.
